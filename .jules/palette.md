# Palette's UX Journal

## 2026-05-22 - Report Scannability Enhancement
**Learning:** For text-heavy compliance reports like the System Security Plan (SSP), users often struggle to quickly identify the overall posture. Adding visual cues like progress bars and status emojis significantly reduces cognitive load and improves the scannability of the document.
**Action:** Use standard Unicode characters for progress bars and consistent emojis for status mappings in Markdown reports.
